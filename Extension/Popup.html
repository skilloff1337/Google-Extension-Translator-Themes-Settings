Translator Themes Setting (DTS)

Translator Themes Setting - This is an open source extension that will allow you to change the color scheme on the Translator website to ready-made ones. And you can also customize the color scheme for yourself by turning on manual settings.

IMPORTANT! If you find a bug or flaw, or with ideas for improving the product, send your messages to the discord (skilloff#4741) or github issues.

If the app turned out to be useful, do not forget to rate it in the extension store.
